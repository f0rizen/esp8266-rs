# esp8266-rs
Example blinky project for ESP8266 using Rust Programming Language

## Flashing
run `flash.sh` script

### Requirements
- `Docker`
- `Linux Distro`
- Script assumes device is `/dev/ttyUSB0` (change if neccessary).
